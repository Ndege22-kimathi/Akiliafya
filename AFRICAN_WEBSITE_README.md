# 🌍 Ubuntu Spirit - African Website

**Karibu! Welcome to Ubuntu Spirit** - A vibrant celebration of African culture, innovation, and community spirit.

## 🎨 Features

This beautiful African-themed website showcases:

- **🎵 Rich Cultural Heritage**: Music, dance, art, crafts, stories, and cuisine
- **🤝 Ubuntu Philosophy**: "I am because we are" - celebrating interconnectedness
- **💡 African Innovation**: Highlighting the continent's technological advancement
- **🌈 Vibrant Design**: Authentic African colors (gold, red, green, orange)
- **📱 Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **✨ Modern Animations**: Smooth scrolling, fade-ins, and interactive elements

## 🚀 Quick Start

### Option 1: Simple Python Server
```bash
# Run the included server script
python3 serve_african_site.py

# Then visit: http://localhost:8000/african_home.html
```

### Option 2: Direct File Access
Simply open `templates/african_home.html` in your web browser.

### Option 3: Django Integration
If you're using the existing Django project:
1. Add a URL pattern to serve the new template
2. Create a view to render `african_home.html`

## 🛠️ Technical Details

- **Framework**: Pure HTML5, CSS3, JavaScript (ES6+)
- **CSS Framework**: Bootstrap 5.3.0
- **Icons**: Bootstrap Icons
- **Fonts**: Google Fonts (Ubuntu, Oswald, Dancing Script)
- **Animations**: CSS animations with Intersection Observer API
- **Responsive**: Mobile-first design approach

## 🌟 Special Features
